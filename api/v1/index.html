Machine-readable endpoints

Every claim on the dashboard is backed by a static JSON artifact. This page is the catalog. All endpoints are CORS-open static files served from GitHub Pages.

Verification chain. Validate snapshot.json against snapshot.schema.json, then verify snapshot.sig with cosign verify-blob --bundle snapshot.sig snapshot.json against the workflow identity in snapshot.sig.meta.json. If either step fails, do not trust the rendered HTML — see EVIDENCE.md.
